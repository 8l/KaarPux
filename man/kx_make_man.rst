kx_make_man - Create KaarPux manual pages
=========================================


SYNOPSIS
--------
**kx_make_man**


DESCRIPTION
-----------
Create **KaarPux** manual pages


FILES
-----

The man page source is found in the :file:`master/man` subdirectory.
The man pages will be created in the :file:`man` subdirectory.
